API Documentation

Integrate Butter Labs into your applications

REST API & WebSocket Integration

Our comprehensive API allows you to integrate Butter Labs' accent-aware conversational capabilities into any application.

Base URL

https://api.butterlabs.ai/v1
POST /api/v1/speech/analyze

Analyze speech input and detect accent

Request Example:
{
  "audio_data": "base64_encoded_audio",
  "format": "wav",
  "sample_rate": 16000
}
Response Example:
{
  "accent": "scottish",
  "confidence": 0.96,
  "text": "Hello, I need help with my account",
  "processing_time_ms": 85
}
GET /api/v1/accents/supported

Get list of supported accents

Response Example:
{
  "accents": [
    "scottish", "welsh", "northern_irish",
    "geordie", "yorkshire", "cornish",
    "brummie", "scouse", "devon",
    "norfolk", "cockney", "rp",
    "west_country", "midlands"
  ],
  "total_count": 14
}

Getting Started

Quick setup and integration guide

1

Sign Up & API Key

Create your Butter Labs account and obtain your API key

curl -X POST https://api.butterlabs.ai/v1/auth/register
2

Install SDK

Choose your preferred programming language and install our SDK

npm install @butterlabs/ai-sdk
3

Configure & Test

Initialize the client and run your first accent recognition test

const client = new ButterLabs({
  apiKey: 'your-api-key'
});

const result = await client.speech.analyze(audioData);
